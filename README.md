# 📱 Telecom Customer Usage Analysis

## 📌 Project Overview

This project analyzes customer behavior for **ConnectaTel**, a telecommunications company operating in Mexico and Colombia.

The objective is to understand how customers use mobile services, identify usage patterns, detect anomalous behavior, and generate actionable business insights that support commercial strategy and improve customer experience.

---

## 🎯 Business Questions

This analysis aims to answer:

- Which customer segments use calls and messages the most?
- Are there customers with unusual behavior that could indicate fraud or data quality issues?
- How does usage vary by age group and mobile plan?
- What opportunities exist to optimize current plans and customer segmentation?

---

---

## 📂 Dataset

The analysis integrates three datasets:

- **plans.csv** – Mobile plans and pricing.
- **users_latam.csv** – Customer demographic information.
- **usage.csv** – Calls and text message activity.

---

## 📊 Methodology

The project followed these stages:

1. Data loading
2. Data quality assessment
3. Data cleaning
4. Exploratory Data Analysis (EDA)
5. Summary statistics
6. Outlier detection
7. Customer segmentation
8. Business insights and recommendations

---

## 🔍 Key Findings

- Customer usage varies significantly across demographic segments.
- Most customers fall into medium-usage categories.
- High-usage customers represent valuable commercial opportunities.
- Several outliers were identified that should be reviewed individually.
- Data quality issues (missing values and invalid records) were successfully addressed before analysis.

---

## 💼 Business Recommendations

- Design premium plans for high-consumption users.
- Personalize offers according to customer age and usage.
- Monitor anomalous behavior continuously.
- Maintain data quality validation before future analyses.
